This is your new *vault*.

Make a note of something, [[create a link]], or try [the Importer](https://help.obsidian.md/Plugins/Importer)!

- Cmd B = bold
- Cmd I = italic
- ~~ = strikethrough
- == = highlight
- "#" = heading
- ">" = quote block
- "-" = bullet list
- "1." = numbered list
- "- [ ] " = checklist (must press space after each of them)
- "---" = divider
- "`" = inline code`
- "```" = code block ```
- "[[]]" = link to note
- "[]()" = link to external site
- Cmd + T = new tab
- Cmd + W = close tab
- ! before link to note or external site embed the note/image from the site.
